Voice Chat FAQ

This website is served by the Brickadia server you're playing on as an Omegga plugin. You can view the source or get the plugin for yourself here.

Firefox issues

There are known issues with Firefox/Gecko browsers. If the voice chat is not working at all, it is suggested that you switch over to a Chromium-based browser, like Google Chrome or Microsoft Edge.

If Firefox is mostly working for you but newly joined users don't seem to be audible to you, refocus your browser tab every now and then. Firefox seems to throttle the JavaScript and sometimes prevents certain things from happening when the browser is not focused.

Verticality

Presently, verticality is only handled as distance- the farther away vertically you are from someone, the quieter they sound. This does not account for pitch (the up-down angle of your player) as Brickadia does not expose this.
